## Todo cli
- A basic todo cli app written in go. Persist the state using json file.
- todo.json file can be found in data directory

### Operations
1. 0 - exit
2. 1 - list all todos
3. 2 - add a todo
4. 3 - toggle status of a todo
5. 4 - update a todo
6. 5 - list completed todos
7. 6 - list uncompeleted todos
8. 8 - clear screen
9. 9 - list commands
